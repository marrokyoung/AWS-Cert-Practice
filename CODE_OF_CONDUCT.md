# Code of Conduct

## Our Standard

This project is intended to be useful, technically serious, and welcoming to contributors acting in good faith.

Participants are expected to:

- Be respectful and direct
- Focus criticism on ideas, code, and evidence
- Assume good intent unless behavior shows otherwise
- Accept correction without turning discussion into conflict
- Keep discussion relevant to the project

## Unacceptable Behavior

The following are not acceptable in project spaces:

- Harassment, intimidation, or personal attacks
- Discriminatory or hateful language
- Deliberate disruption, trolling, or bad-faith argument
- Publishing private information without consent
- Repeatedly ignoring project rules, review guidance, or moderation decisions

## Scope

This applies to repository discussions, issues, pull requests, code review, and other project-managed communication spaces.

## Enforcement

The maintainer may remove comments, close discussions, reject contributions, or block users whose behavior harms the project.

## Reporting

If you experience or observe unacceptable behavior, contact [@marrokyoung](https://github.com/marrokyoung). For security issues, use [SECURITY.md](./SECURITY.md) instead of public discussion.
